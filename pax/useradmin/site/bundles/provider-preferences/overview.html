Preferences Provider Bundle

The Preferences Provider implementation of the StorageProvider SPI uses the OSGi Preferences service as backend storage for persisting user data.

The layout of the storage is compatible to the layout used by the Equinox UserAdmin implementation.

See also:

  • Preferences Configuration and the quickstart guide
